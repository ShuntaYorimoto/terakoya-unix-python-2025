## 1. BLASTとは

**BLAST**（Basic Local Alignment Search Tool）は、配列データベースの中からクエリ配列に類似した配列を高速に検索するツールです。バイオインフォマティクスで最も広く使われているツールの1つです。

| プログラム | クエリ | データベース | 用途 |
|-----------|--------|------------|------|
| `blastn` | 塩基配列 | 塩基配列 | DNA/RNA配列の検索 |
| `blastp` | アミノ酸配列 | アミノ酸配列 | タンパク質配列の検索 |
| `blastx` | 塩基配列（6フレーム翻訳） | アミノ酸配列 | DNA配列をタンパク質DBで検索 |
| `tblastn` | アミノ酸配列 | 塩基配列（6フレーム翻訳） | タンパク質配列を塩基配列DBで検索 |
| `tblastx` | 塩基配列（6フレーム翻訳） | 塩基配列（6フレーム翻訳） | 翻訳レベルでの塩基配列同士の検索 |

BLASTのアルゴリズムの詳細は本講義では扱いません。ここでは、コマンドラインでBLASTを実行する方法を体験しましょう。

---

## 2. BLASTデータベースの作成

BLASTは、検索を高速に行うためにFASTAファイルを専用のデータベース形式に変換する必要があります。

```bash
$ makeblastdb -in database.fasta -dbtype prot -out db
```

| オプション | 説明 |
|-----------|------|
| `-in` | 入力FASTAファイル |
| `-dbtype` | データベースの種類（`prot`: タンパク質、`nucl`: 塩基配列） |
| `-out` | 出力データベース名（プレフィックス） |

実行すると、`db.phr`、`db.pin`、`db.psq` などのファイルが生成されます。

## 3. BLASTPの実行

### デフォルト出力

```bash
$ blastp -query query.fasta -db db -out query_vs_db.txt
```

| オプション | 説明 |
|-----------|------|
| `-query` | クエリ配列（FASTAファイル） |
| `-db` | 検索対象のデータベース名 |
| `-out` | 出力ファイル名 |

### タブ区切り出力

```bash
$ blastp -query query.fasta -db db -out query_vs_db.fmt6.txt -outfmt 6
```

| オプション | 説明 |
|-----------|------|
| `-outfmt 6` | タブ区切り形式で出力（BLAST tabular format） |

## 4. 結果の確認

### デフォルト出力の確認

```bash
$ less query_vs_db.txt
```

デフォルト出力には以下の情報が含まれます：

- **ヒットリスト**: 類似配列の一覧（スコア順）
- **アライメント**: クエリとヒット配列の詳細な並び
- **統計情報**: E値（偶然に類似する確率の指標）やスコア

### タブ区切り出力の確認

```bash
$ less query_vs_db.fmt6.txt
```

タブ区切り形式（`-outfmt 6`）では、各ヒットが1行にまとまっており、列ごとに情報が整理されています。
| 列番号 | フィールド名 | 説明 |
|--------|------------|------|
| 1 | qseqid | クエリ配列のID |
| 2 | sseqid | ヒット配列のID |
| 3 | pident | 一致率（%） |
| 4 | length | アライメントの長さ |
| 5 | mismatch | ミスマッチ数 |
| 6 | gapopen | ギャップ開始数 |
| 7 | qstart | クエリの開始位置 |
| 8 | qend | クエリの終了位置 |
| 9 | sstart | ヒットの開始位置 |
| 10 | send | ヒットの終了位置 |
| 11 | evalue | E値（小さいほど有意） |
| 12 | bitscore | ビットスコア（大きいほど有意） |

---

## 6. 出力フォーマットの違い

| 項目 | デフォルト出力 | タブ区切り出力 (`-outfmt 6`) |
|------|--------------|---------------------------|
| 形式 | 人間が読みやすいテキスト | プログラムで処理しやすい表形式 |
| アライメント表示 | あり | なし |
| 統計情報の詳細 | あり | 行ごとに要約 |
| 用途 | 結果を目で確認するとき | 後のセクションで学ぶコマンドで集計するとき |

---

## まとめ

| コマンド | 説明 |
|---------|------|
| `makeblastdb` | FASTAファイルからBLASTデータベースを作成 |
| `blastp` | タンパク質配列の類似性検索 |
| `-outfmt 6` | タブ区切り形式で結果を出力 |

タブ区切り出力（`-outfmt 6`）は列ごとに情報が整理されているため、後のセクションで学ぶテキスト処理コマンドと組み合わせることで、ヒットの絞り込みや集計を効率的に行えます。
