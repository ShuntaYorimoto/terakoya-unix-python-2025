## 問1：リストの操作

以下は ORF（Open Reading Frame）のコドンリストです。

```python
codons = ["ATG", "GCT", "GGT", "ATG", "TAA", "TGG", "ATG"]
```

1. リストを**アルファベット順**（昇順）に並べ替えて出力してください
2. `in` を使って `"TAA"`（終止コドン）が含まれているか確認して出力してください
3. `.count()` を使って `"ATG"`（開始コドン）が何個あるか数えて出力してください

**出力例：**

```
['ATG', 'ATG', 'ATG', 'GCT', 'GGT', 'TAA', 'TGG']
TAA found: True
ATG count: 3
```

<details>
<summary>解答例</summary>

```python
codons = ["ATG", "GCT", "GGT", "ATG", "TAA", "TGG", "ATG"]
print(sorted(codons))
print(f"TAA found: {'TAA' in codons}")
print(f"ATG count: {codons.count('ATG')}")
```

</details>

## 問2：辞書によるコドン検索

以下のコドン→アミノ酸辞書があります。

```python
codon_table = {"ATG": "Met", "TAA": "Stop", "GCT": "Ala", "GGT": "Gly"}
```

1. `"ATG"` に対応するアミノ酸を辞書から取得して出力してください
2. `"TGG"` を `.get()` で検索し、辞書にない場合は `"Unknown"` と出力してください
3. `"TAA"` が辞書のキーに存在するか `in` を使って確認して出力してください

**出力例：**

```
ATG -> Met
TGG -> Unknown
TAA in table: True
```

<details>
<summary>解答例</summary>

```python
codon_table = {"ATG": "Met", "TAA": "Stop", "GCT": "Ala", "GGT": "Gly"}

print(f"ATG -> {codon_table['ATG']}")
print(f"TGG -> {codon_table.get('TGG', 'Unknown')}")
print(f"TAA in table: {'TAA' in codon_table}")
```

> `dict.get(key, default)` はキーが存在しない場合に `default` を返します。`codon_table['TGG']` のように直接アクセスすると `KeyError` が発生するので注意。

</details>

## 問3：集合演算で遺伝子を比較する

Sample A と Sample B でそれぞれ発現が検出された遺伝子のリストが以下のように与えられています。

```python
sample_a = ["atpA", "atpB", "atpD", "leuA", "trpG", "groL", "dnaK"]
sample_b = ["atpA", "atpB", "groL", "dnaK", "rplA", "rpsA", "infA"]
```

以下の3つを求めて出力してください。

1. **両方のサンプルに共通する**遺伝子（積集合）
2. **Sample A のみ**に存在する遺伝子（差集合）
3. **Sample B のみ**に存在する遺伝子（差集合）

**出力例：**（順序は問いません）

```
Common: {'atpA', 'atpB', 'dnaK', 'groL'}
Only in A: {'atpD', 'leuA', 'trpG'}
Only in B: {'infA', 'rplA', 'rpsA'}
```

<details>
<summary>解答例</summary>

```python
sample_a = ["atpA", "atpB", "atpD", "leuA", "trpG", "groL", "dnaK"]
sample_b = ["atpA", "atpB", "groL", "dnaK", "rplA", "rpsA", "infA"]

set_a = set(sample_a)
set_b = set(sample_b)

print("Common:", set_a & set_b)
print("Only in A:", set_a - set_b)
print("Only in B:", set_b - set_a)
```

</details>
