## 問題2：ファイルとディレクトリを判定するスクリプトを作成する

`for` ループと `if/elif` を組み合わせて、アイテムの種類（ファイルかディレクトリか）を判定するスクリプトを書きます。`04_shell_script.md` の `for` と `if` の例を参考にしましょう。

- スクリプト名は `check_items.sh` とする
- `ITEMS=../*` のように変数でループ対象を指定し、`for ITEM in ${ITEMS}` でループする
- `if [ -f ... ]` の条件が真なら `"${ITEM} is a file."` と表示する
- `elif [ -d ... ]` の条件が真なら `"${ITEM} is a directory."` と表示する
- `else` の場合は `"${ITEM} is something else."` と表示する
- `bash check_items.sh` を実行し、`~/work/` 以下のディレクトリとファイルが判定されることを確認する
