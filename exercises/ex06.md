## 問1：ゲノムサイズの単位変換

ササコナフキツノアブラムシのブフネラのゲノムサイズは **432,286 bp** です。
これを **Mbp（メガ塩基対）** 単位に変換し、小数点以下2桁まで表示してください。

**出力例：**

```
Genome size: 0.43 Mbp
```

<details>
<summary>解答例</summary>

```python
genome_bp = 432286
genome_mbp = genome_bp / 1000000
print(f"Genome size: {genome_mbp:.2f} Mbp")
```

</details>

## 問2：アミノ酸組成の計算

以下は エンドウヒゲナガアブラムシ *Acyrthosiphon pisum* の共生器官に高発現する遺伝子 *BCR1* (Bacteriocyte-specific Cysteine-Rich protein 1)のアミノ酸配列です(Shigenobu and Stern, 2013)。

```python
bcr1 = "MKLLHGFLIIMLTMHLSIQYAYGGPFLTKYLCDRVCHKLCGDEFVCSCIQYKSLKGLWFPHCPTGKASVVLHNFLTSP"
```

`len()` と `str.count()` を使って、この配列の**アミノ酸数**と **Cys（`C`）の数および割合**を f-string で出力してください。

**出力例：**

```
Amino acids: 78
Cys count: 6
Cys fraction: 7.69%
```

<details>
<summary>解答例</summary>

```python
bcr1 = "MKLLHGFLIIMLTMHLSIQYAYGGPFLTKYLCDRVCHKLCGDEFVCSCIQYKSLKGLWFPHCPTGKASVVLHNFLTSP"

total = len(bcr1)
cys_count = bcr1.count("C")
cys_frac = cys_count / total * 100
print(f"Amino acids: {total}")
print(f"Cys count: {cys_count}")
print(f"Cys fraction: {cys_frac:.2f}%")
```

</details>

## 問3：FASTA ヘッダーから Protein ID を取り出す

以下の FASTA ヘッダー文字列から、**Protein ID**（`PAX6_HUMAN`）を取り出して出力してください。
`split("|")` などを使ってください。

```python
header = ">sp|P26367|PAX6_HUMAN Paired box protein Pax-6 OS=Homo sapiens OX=9606 GN=PAX6 PE=1 SV=1"
```

**出力例：**

```
Protein ID: PAX6_HUMAN
```

<details>
<summary>解答例</summary>

```python
header = ">sp|P26367|PAX6_HUMAN Paired box protein Pax-6 OS=Homo sapiens OX=9606 GN=PAX6 PE=1 SV=1"

parts = header[1:].split("|")
protein_id = parts[2].split()[0]
print(f"Protein ID: {protein_id}")
```

</details>
