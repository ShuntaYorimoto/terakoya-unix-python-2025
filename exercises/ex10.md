## 問1：FASTA ファイルの読み込みと情報表示

`SeqIO.parse()` を使って `pax_human.fasta` を読み込み、各タンパク質の **ID** と**アミノ酸数**を出力してください。最後に配列の総数も出力してください。

**出力例：**

```
sp|P26367|PAX6_HUMAN : 422 aa
sp|Q02548|PAX5_HUMAN : 391 aa
sp|Q02962|PAX2_HUMAN : 417 aa
sp|Q06710|PAX8_HUMAN : 450 aa
合計: 4 配列
```

> `SeqIO.parse(ファイルパス, "fasta")` は各レコードを `SeqRecord` オブジェクトとして返します。
> `record.id` で ID、`len(record.seq)` でアミノ酸数が得られます。

<details>
<summary>解答例</summary>

```python
from Bio import SeqIO

fasta_file = "../ex09/pax_human.fasta"

count = 0
for record in SeqIO.parse(fasta_file, "fasta"):
    print(f"{record.id} : {len(record.seq)} aa")
    count += 1

print(f"合計: {count} 配列")
```

</details>

## 問2：条件を満たす配列を別ファイルに書き出す

`pax_human.fasta` から、アミノ酸数が **420 aa 以上**のタンパク質配列だけを選び出して `~/work/ex10/large_pax.fasta` に書き出してください。

書き出し後、何件の配列が書き出されたか出力してください。

**出力例：**

```
Large proteins (>= 420 aa): 2 sequences written to large_pax.fasta
```

<details>
<summary>ヒント</summary>

- `SeqIO.parse()` で読み込んだレコードを `for` ループで処理する
- 条件を満たすレコードをリストに追加していく：

```python
input_file = "../ex09/pax_human.fasta"

large = []
for record in SeqIO.parse(input_file, "fasta"):
    if len(record.seq) >= 420:
        large.append(record)
```

- `SeqIO.write(レコードのリスト, ファイルパス, "fasta")` でまとめて書き出せる
- 書き出した件数は `len(large)` で確認できる

</details>

<details>
<summary>解答例</summary>

```python
from Bio import SeqIO

input_file = "../ex09/pax_human.fasta"
output_file = "large_pax.fasta"

large = []
for record in SeqIO.parse(input_file, "fasta"):
    if len(record.seq) >= 420:
        large.append(record)

SeqIO.write(large, output_file, "fasta")
print(f"Large proteins (>= 420 aa): {len(large)} sequences written to {output_file}")
```

</details>
