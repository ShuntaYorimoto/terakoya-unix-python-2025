## 問1：FASTA ファイルの読み込み

`pax_human.fasta` を読み込み、**ヘッダー行**の内容（`>` を除いたもの）を表示してください。最後にエントリの総数も出力してください。

**出力例：**

```
sp|P26367|PAX6_HUMAN Paired box protein Pax-6 OS=Homo sapiens OX=9606 GN=PAX6 PE=1 SV=2
sp|Q02548|PAX5_HUMAN Paired box protein Pax-5 OS=Homo sapiens OX=9606 GN=PAX5 PE=1 SV=1
sp|Q02962|PAX2_HUMAN Paired box protein Pax-2 OS=Homo sapiens OX=9606 GN=PAX2 PE=1 SV=4
sp|Q06710|PAX8_HUMAN Paired box protein Pax-8 OS=Homo sapiens OX=9606 GN=PAX8 PE=1 SV=2
エントリ数: 4
```

> - 各行末の改行文字 `\n` を取り除くには `.strip()` を使います。
> - `>` で始まる行がヘッダー行です。`startswith(">")` で判定できます。

<details>
<summary>解答例</summary>

```python
count = 0

with open("pax_human.fasta") as f:
    for line in f:
        line = line.strip()
        if line.startswith(">"):
            print(line[1:])    # ">" を除去して表示
            count += 1

print(f"エントリ数: {count}")
```

</details>

## 問2：BLAST 結果の集計

`eyeless_vs_human.fmt6.txt` を読み込み、**Evalueが `1e-50` 以下**の行のヒット名を表示してください。合計件数も出力してください。

**出力例：**

```
sp|P26367|PAX6_HUMAN
sp|Q02548|PAX5_HUMAN
sp|Q02962|PAX2_HUMAN
sp|Q06710|PAX8_HUMAN
sp|O43316|PAX4_HUMAN
sp|P55771|PAX9_HUMAN
ヒット数: 6
```

> - `line.split("\t")` で各行をタブで分割するとフィールドのリストが得られます。

<details>
<summary>解答例</summary>

```python
count = 0

with open("eyeless_vs_human.fmt6.txt") as f:
    for line in f:
        fields = line.split("\t")
        evalue = float(fields[10])
        if evalue <= 1e-50:
            print(fields[1])
            count += 1

print(f"ヒット数: {count}")
```

</details>
