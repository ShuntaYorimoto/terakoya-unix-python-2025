## 問1：PHRED クオリティスコアの判定

FASTQ ファイルの塩基クオリティは PHRED スコアで表されます。以下の変数が与えられています。

```python
phred_score = 28
```

`phred_score` の値に応じて、次の基準でメッセージを出力する `if / elif / else` 文を書いてください。

| 条件 | メッセージ |
|------|-----------|
| 30 以上 | `"High quality (Q30+)"` |
| 20 以上 30 未満 | `"Acceptable quality"` |
| 20 未満 | `"Low quality — consider filtering"` |

**出力例：**

```
Q28: Acceptable quality
```

<details>
<summary>解答例</summary>

```python
phred_score = 28

if phred_score >= 30:
    message = "High quality (Q30+)"
elif phred_score >= 20:
    message = "Acceptable quality"
else:
    message = "Low quality — consider filtering"

print(f"Q{phred_score}: {message}")
```

</details>

## 問2：コドン位置の出力

DNA 配列をコドン単位（3塩基ずつ）で読み、各コドンが開始コドン・終止コドン・それ以外かを判定して出力してください。

```python
seq = "ATGCGATAAATGGTTTAA"
```

`range(0, len(seq), 3)` を使って先頭位置をループし、`seq[i:i+3]` でコドンを取り出してください。

判定基準：
- `ATG` → `[start]`
- `TAA`, `TAG`, `TGA` → `[stop]`
- それ以外 → `[   ]`

**出力例：**

```
pos  0: ATG [start]
pos  3: CGA [   ]
pos  6: TAA [stop]
pos  9: ATG [start]
pos 12: GTT [   ]
pos 15: TAA [stop]
```

<details>
<summary>解答例</summary>

```python
seq = "ATGCGATAAATGGTTTAA"

for i in range(0, len(seq), 3):
    codon = seq[i:i+3]
    if codon == "ATG":
        label = "[start]"
    elif codon in ["TAA", "TAG", "TGA"]:
        label = "[stop] "
    else:
        label = "[   ]  "
    print(f"pos {i:2d}: {codon} {label}")
```

</details>

## 問3：相補塩基を返す関数

DNA の1塩基を受け取り、その**相補塩基**を返す関数 `complement_base(base)` を `def` と `return` を使って定義してください。

| 入力 | 出力 |
|------|------|
| `"A"` | `"T"` |
| `"T"` | `"A"` |
| `"G"` | `"C"` |
| `"C"` | `"G"` |

定義後、以下のコードで動作を確認してください。

```python
for base in ["A", "T", "G", "C"]:
    print(f"{base} -> {complement_base(base)}")
```

**出力例：**

```
A -> T
T -> A
G -> C
C -> G
```

<details>
<summary>解答例</summary>

```python
def complement_base(base):
    if base == "A":
        return "T"
    elif base == "T":
        return "A"
    elif base == "G":
        return "C"
    elif base == "C":
        return "G"

for base in ["A", "T", "G", "C"]:
    print(f"{base} -> {complement_base(base)}")
```

</details>
