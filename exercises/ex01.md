# 復習問題01：UNIX初級

本演習では、ササコナフキツノアブラムシ *Ceratovacuna japonica* の共生細菌ブフネラ *Buchnera aphidicola* のゲノムデータ（Yorimoto et al., 2022, *iScience*）を使用します。

---

## 準備：データのダウンロードと配置

以下のコマンドをターミナルに **1行ずつ** 入力して実行してください。コメント行は実行する必要はありません。

```bash
# 演習データのアーカイブをダウンロードする
wget https://github.com/ShuntaYorimoto/terakoya-unix-python-2025/blob/main/data.tar.gz

# アーカイブを展開する（-x: 展開, -z: gzip, -f: ファイル指定）
tar -xzf data.tar.gz

# 作業ディレクトリを作成する（-p: 中間ディレクトリも含めて作成）
mkdir -p ~/work/ex01

# データファイルを作業ディレクトリにコピーする
cp ~/data/ex01/* ~/work/ex01/

# 作業ディレクトリに移動する
cd ~/work/ex01

# 現在のディレクトリを確認する
pwd

# ファイルが揃っているか確認する
ls -lh
```

**確認：** 以下の3つのファイルが表示されれば準備完了です。

```
buchnera_cj.fna      # ゲノム塩基配列（FASTA形式）
buchnera_cj.faa.gz   # タンパク質配列（FASTA形式、gzip圧縮済み）
buchnera_cj.gff   # ゲノムアノテーション（GFF3形式）
```
