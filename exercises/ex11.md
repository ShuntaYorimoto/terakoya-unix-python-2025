## 問1：NumPy による基本統計

以下の配列は、ある実験で測定した遺伝子の発現量（RPKM）を表しています。

```python
expression = np.array([32, 45, 28, 51, 39, 47, 33, 60, 22, 41])
```

この配列の**平均・最大値・最小値・標準偏差**を計算して出力してください。

**出力例：**

```
Mean:   39.80
Max:    60
Min:    22
Std:    10.85
```

<details>
<summary>解答例</summary>

```python
import numpy as np

expression = np.array([32, 45, 28, 51, 39, 47, 33, 60, 22, 41])

print(f"Mean:   {np.mean(expression):.2f}")
print(f"Max:    {np.max(expression)}")
print(f"Min:    {np.min(expression)}")
print(f"Std:    {np.std(expression):.2f}")
```

</details>

## 問2：BLAST 結果を Pandas で読み込む

`pd.read_csv()` を使って `../ex09/eyeless_vs_human.fmt6.txt` を読み込んでください。
`-outfmt 6` の出力には列名がないので、以下のように手動で指定します。

```python
columns = ["qseqid", "sseqid", "pident", "length", "mismatch",
           "gapopen", "qstart", "qend", "sstart", "send", "evalue", "bitscore"]
```

読み込み後、**evalue < 1e-10** の行を抽出して件数を表示してください。

**出力例：**

```
Total hits: 253
Hits with evalue < 1e-10: 64
```

> 列名がないファイルは `header=None` を指定します。代わりに、自分で用意した列名を使うには `names=columns` で指定します。

<details>
<summary>解答例</summary>

```python
import pandas as pd

blast_file = "../ex09/eyeless_vs_human.fmt6.txt"
columns = ["qseqid", "sseqid", "pident", "length", "mismatch",
           "gapopen", "qstart", "qend", "sstart", "send", "evalue", "bitscore"]

df = pd.read_csv(blast_file, sep="\t", header=None, names=columns)

filtered = df[df["evalue"] < 1e-10]

print(f"Total hits: {len(df)}")
print(f"Hits with evalue < 1e-10: {len(filtered)}")
```

</details>

## 問4：DEG 結果とアノテーションのマージ

`deg_results.tsv` と `annotation.tsv` を読み込み、`gene_id` をキーに結合してください。
結合後、**FDR < 0.05** の行を `logFC` 降順に並べ替えて、`gene_id`・`logFC`・`FDR`・`Description` を表示してください。

**出力例：**

```
  gene_id  logFC    FDR                      Description
    leuA    3.1  0.010         2-isopropylmalate synthase
    mnmG    2.3  0.002       tRNA modification enzyme MnmG
    gyrB   -1.8  0.040      DNA topoisomerase subunit B
    atpA   -2.9  0.001          ATP synthase subunit alpha
```

<details>
<summary>解答例</summary>

```python
deg_file = "deg_results.tsv"
annot_file = "annotation.tsv"

deg = pd.read_csv(deg_file, sep="\t")
annot = pd.read_csv(annot_file, sep="\t")

merged = deg.merge(annot, on="gene_id", how="left")

sig = merged[merged["FDR"] < 0.05].sort_values("logFC", ascending=False)
print(sig[["gene_id", "logFC", "FDR", "Description"]].to_string(index=False))
```

> **`to_string(index=False)` とは**：DataFrame を文字列として整形して表示するメソッドです。`index=False` を指定すると行番号（インデックス）が非表示になります。
> 試しに `.to_string(index=False)` を外して `print(sig[...])` だけにすると、行番号が表示される点と出力の見た目がどう変わるか確かめてみましょう。

</details>
